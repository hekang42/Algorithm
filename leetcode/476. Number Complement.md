# Leetcode

## 476. Number Complement 


[문제링크](https://leetcode.com/problems/number-complement/)

: 2의 보수 구기


### 코드

```swift
class Solution {
    func findComplement(_ num: Int) -> Int {
        let maxnum : Int = 2 << (String(num, radix: 2).count - 1)
        
        return maxnum - num - 1
    }
}
```
